# Canopy Documentation

This directory contains the split documentation for Canopy IIIF.

- [Content Authoring](canopy_content.md): Guides on writing MDX, frontmatter usage, and site layout.
- [Developer Guide](canopy_developers.md): Architecture overview, custom component registration, and analytics/SEO.
- [User Guides](canopy_guides.md): Step-by-step walkthroughs for common tasks and workflows.
